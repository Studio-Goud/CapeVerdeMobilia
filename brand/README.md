# Djarvista — brand assets

Official logo files. Concept 01 wordmark: the dot on the **i** is a rising coral sun.

## Colours
| Rol | Hex | Gebruik |
|-----|-----|---------|
| Vlagblauw (Cape Verde flag blue) | `#003893` | woordmerk, badge-achtergrond, primair |
| Koraal (zon) | `#f2612a` | de zon-tittel, accenten |
| Wit | `#ffffff` | tekst/logo op donkere of blauwe achtergrond |

## Bestanden
| Bestand | Wat | Waarvoor |
|---------|-----|----------|
| `djarvista-logo-horizontal.png` | badge + woordmerk (blauw), transparant | website-header, documenten, e-mailhandtekening |
| `djarvista-wordmark.png` | alleen woordmerk (blauw), transparant | op witte/lichte achtergrond |
| `djarvista-wordmark-white.png` | alleen woordmerk (wit), transparant | op donkere/foto-achtergrond |
| `djarvista-icon.svg` / `djarvista-icon.png` | badge "D" met zon | favicon, app-icoon, kleine avatar |
| `djarvista-avatar.png` (1080×1080) | D + zon op vlagblauw, vol vlak | **Instagram / social profielfoto** — veilig binnen een ronde uitsnede |
| `djarvista-social-square.png` (1080×1080) | lockup gecentreerd op wit | vierkante social post / aankondiging |

## Richtlijnen (kort)
- Houd rondom het logo vrije ruimte ≈ de hoogte van de badge.
- Verklein het woordmerk niet zodat de zon-tittel kleiner wordt dan ~10 px.
- Verander de kleuren niet; gebruik de witte variant op donkere achtergronden.
- De PNG's zijn hoge-resolutie en transparant. Voor grootformaat drukwerk kan het
  woordmerk als vector worden aangeleverd (tekst omzetten naar contouren in een
  vectorprogramma) — vraag daar gerust om.

De woordmerken zijn gezet in een vette grotesk (Arial/Helvetica-stijl).
